# FoodGo Login & Signup UI

## Overview
FoodGo Login & Signup UI is a modern authentication prototype designed in Figma for a food delivery mobile application. The project focuses on creating a simple, secure, and user-friendly onboarding experience.

## Features
- Welcome Screen
- User Registration
- Sign In
- Social Login
- Phone Verification
- OTP Verification
- Forgot Password
- Reset Password
- Language Selection

## Tools Used
- Figma
- Google Fonts
- Material Icons
